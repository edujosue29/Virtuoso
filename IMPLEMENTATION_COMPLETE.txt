================================================================================
                         MAINTENANCE & SUPPORT
================================================================================

For Deployment:
  1. Copy dist/ folder to hosting
  2. All translations bundled (no extra files needed)
  3. No environment variables required
  4. Cache-bust on new builds

For Updates:
  1. Adding new translations: Update both es.json and en.json
  2. New components: Add useTranslation() hook, use t() calls
  3. New sections: Create translation keys, update JSON files
  4. Testing: Verify in both languages before merging

Troubleshooting:
  - See BILINGUAL_DEVELOPER_GUIDE.md for common issues
  - Check browser console for i18next warnings
  - Verify translation keys exist in both JSON files
  - Ensure all components have useTranslation() hook

Support Contact:
  For issues or questions, refer to:
  1. BILINGUAL_DEVELOPER_GUIDE.md - Troubleshooting section
  2. i18next documentation: https://www.i18next.com
  3. react-i18next documentation: https://react.i18next.com
